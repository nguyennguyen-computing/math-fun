# math-fun

A fun math game with Firebase authentication and leaderboard.

## Setup

1. Copy `firebase-config.example.js` to `firebase-config.js`
2. Add your Firebase credentials to `firebase-config.js`
3. Open `index.html` in a browser

## Deploy

### Option 1: Netlify
1. Sign up at netlify.com
2. Drag and drop the source-code folder
3. Add Firebase config as environment variables in Netlify dashboard

### Option 2: Vercel
1. Sign up at vercel.com
2. Import your GitHub repository
3. Add Firebase config as environment variables

### Option 3: GitHub Pages
1. Go to repository Settings > Pages
2. Select branch and folder
3. Note: Firebase config will be visible in source code
